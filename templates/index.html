🤖 How It Works

The Flask backend trains ML models at startup and serves predictions via a REST API — no external services needed.

🤖 AI/ML Pipeline

A Random Forest Classifier (120 trees) is trained at startup on 2,000 data points for flood and crop risk classification. A Linear Regression model maps temperature to monthly rainfall. All predictions are served through a Flask REST API at POST /api/predict. A dedicated POST /api/advisory endpoint powers the full Farmer Advisory with crop cards, irrigation schedules, pest alerts, and soil health scoring.
